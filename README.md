## üìä Overview

This project features a comprehensive sales dashboard built using **Microsoft Power BI** to analyze and visualize key performance indicators (KPIs) from a sales dataset. The goal is to provide actionable insights into sales trends, profitability, and customer behavior across various categories and regions.

## ‚ú® Key Features of the Dashboard

The Power BI dashboard includes visualizations and metrics covering:

* **Sales and Profitability:** Analysis of total sales and profit margins over time.
* **Regional Performance:** Sales and profit breakdown by country/region and state/province.
* **Product Analysis:** Identifying top-performing and underperforming products, categories, and sub-categories.
* **Customer/Segment Analysis:** Segmentation of sales by customer type (e.g., Consumer, Corporate, Home Office).
* **Shipping Mode Impact:** Analyzing the effect of different shipping modes on delivery time and cost.

## üíæ Data Source

The analysis is based on a single dataset:

* **`Sales Overview Data.xlsx - Sales Data.csv`**: Contains detailed transaction-level data including order information, product details, sales figures, quantity, discount, and profit.
